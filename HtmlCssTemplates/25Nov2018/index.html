Interfaces don’t implement another interface, they extend it. Since interfaces can’t have method implementations, there is no issue of diamond problem.it’s mandatory to provide implementation of the method in the class implementing them. For more details with examples

From Java 8 onwards, interfaces can have default method implementations. So to handle diamond problem when a common default method is present in multiple interface

That’s why we have multiple inheritance in interfaces i.e an interface can extend multiple interfaces.
